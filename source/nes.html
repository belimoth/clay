    NES Architecture Overview
    • The CPU reads code and data from the cartridge
    • The CPU reads button input from attached gamepads
    • The CPU writes and reads variables to and from WRAM
    • The CPU writes sound & music data to the APU
    • The CPU writes sprite data to OAM, via the PPU
    • The CPU writes background data to VRAM, via the PPU
    • The PPU reads sprite data from OAM
    • The PPU reads background data from VRAM
    • The PPU reads graphics data from the cartridge
    • The PPU renders video output
    • The APU renders audio output
